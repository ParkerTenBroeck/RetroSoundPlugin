# RetroSoundPlugin

A plugin for the MIPS Virtual processor / assembler to add Famicon sound emulation

## Supported sound chips

### Always Enabled
- 2A03
### Optional/Add-ons - (not implemented yet)
- FDS
- MMC5
- Sunsoft 5B
- VRC6
- VRC7
